# tutorials

## preparation
* download and install the latest Werckmeister compiler from here: https://werckme.github.io/getwerckmeister
* download this soundfont file https://github.com/urish/cinto/blob/master/media/FluidR3%20GM.sf2 and copy it into your werckmeister workspace folder root
* download and install VSCode: https://code.visualstudio.com/
* start VSCode and install the Werckmeister extension
